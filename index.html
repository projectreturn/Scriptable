Scriptable Import in 3 Schritten

Schritt 1: Scriptable aus dem App Store laden

Scriptable im App Store öffnen

Schritt 2: Source-Code kopieren

Klicke auf den Button, um den Script-Code in die Zwischenablage zu kopieren.

// Scriptable Script: SRA-Medium.js

(async () => {
  // Hier steht dein Script-Code, als Beispiel ein minimaler Dummy
  // Ersetze diesen Text durch den kompletten Inhalt deiner JS-Datei

  let alert = new Alert();
  alert.title = "Hallo Scriptable!";
  alert.message = "Das ist ein Beispiel-Script.";
  alert.addAction("OK");
  await alert.present();
})();
Code kopiert! Jetzt Scriptable öffnen.

Schritt 3: Script in Scriptable anlegen

Öffne Scriptable auf deinem iPhone, erstelle ein neues Script und füge den kopierten Code dort ein.

Tipp: Lange auf das "+" in Scriptable drücken, um ein neues Script zu erstellen, dann Code einfügen.
